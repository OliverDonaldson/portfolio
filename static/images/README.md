# Image Directory

Place your website images in this directory:

## Required Images:

### Hero Section (Slideshow)
- `hero1.jpg` - First hero image (1920x500px recommended)
- `hero2.jpg` - Second hero image (1920x500px recommended)  
- `hero3.jpg` - Third hero image (1920x500px recommended)

### Profile
- `profile.jpg` - Your profile photo (500x500px recommended, will be cropped to circle)

### Projects
- `project1.jpg` - Thumbnail for first project (400x300px recommended)
- `project2.jpg` - Thumbnail for second project (400x300px recommended)
- `project3.jpg` - Thumbnail for third project (400x300px recommended)
- `project4.jpg` - Thumbnail for fourth project (400x300px recommended)

### Certificates
- `cert1.jpg` - First certificate image
- `cert2.jpg` - Second certificate image
- `cert3.jpg` - Third certificate image
- `cert4.jpg` - Fourth certificate image

### Activities
- `activity1.jpg` - First activity image (800x600px recommended)
- `activity2.jpg` - Second activity image (800x600px recommended)

### Project Detail Images (for individual project pages)
- `analysis_chart1.png` - Example chart for data analysis project

## Image Guidelines:
- Use JPEG format for photos, PNG for graphics with transparency
- Optimize images for web (compress to reduce file size)
- Use consistent aspect ratios for similar image types
- Ensure images are high quality but not too large (< 1MB each recommended)
